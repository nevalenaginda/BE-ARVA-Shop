## üíª Installation

Follow the steps below

1. Clone this repo
```
git clone https://github.com/nevalenaginda/BE-ARVA-Shop
cd BE-ARVA-Shop
```

2. Install module & Import Database
##### Install Module
```
npm install
```

3. Create env file
```
# ---------------------------------------
#               CONFIG DB
# ---------------------------------------
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
DB_DIALECT=

# ---------------------------------------
#            CONFIG WEBSITE
# ---------------------------------------
DOMAIN=

# ---------------------------------------
#            CONFIG MAILER
# ---------------------------------------
EMAIL=
PASSWORD=

# ---------------------------------------
#            CONFIG GENERAL
# ---------------------------------------
PORT=
HOST=
SECRET_KEY=
SECRET_KEY_RESET=
SECRET_KEY_VERIF=
SECRET_KEY_REFRESH=
SECRET_KEY_PIN=
STATIC_FOLDER=/avatar
```
Detail CONFIG GENERAL
| EXAMPLE URL | [http://localhost:6000]() |
| :-------------: |:-------------:|
| PORT | 6000 |
| HOST | [http://localhost]() |

4. Done, You can run it in the way below
##### Developer Mode (with nodemon)
```
npm run dev
```
##### Production Mode (only node)
```
npm start
```
